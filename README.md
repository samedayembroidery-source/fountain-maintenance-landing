# Fountain Maintenance Inc - Landing Page

Professional fountain maintenance, repairs, and installation website.

## Getting Started

### Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Deployment

1. Push to GitHub
2. Connect GitHub repo to Vercel
3. Add domain in Vercel settings
4. Deploy

## Tech Stack

- Next.js 14
- React 18
- Pure CSS (no build tools needed)

## Features

- Responsive design
- Service carousel
- Pricing plans
- Contact form with lead capture
- Mobile-optimized
